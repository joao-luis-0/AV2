# ✈️ AeroCode — Sistema de Gerenciamento e Controle de Produção de Aeronaves

> Plataforma web para gerenciamento do ciclo de produção de aeronaves, desenvolvida com Node.js e Express.

---

## 📋 Sobre o Projeto

O **AeroCode** é um sistema interno voltado para equipes de empresas aeronáuticas que precisam acompanhar e controlar o processo de fabricação de aeronaves. O sistema centraliza o controle de aeronaves, peças, etapas de produção, testes e equipe em uma única plataforma.

---

## 🖥️ Telas do Sistema

| Tela | Descrição |
|------|-----------|
| **Login** | Autenticação por email e senha |
| **Dashboard** | Visão geral das aeronaves cadastradas com pesquisa em tempo real |
| **Detalhes da Aeronave** | Modal com informações completas e geração de relatório |
| **Cadastro de Peças** | Registro de peças com fabricante e status |
| **Cadastro de Etapas** | Registro de etapas vinculadas a uma aeronave |
| **Cadastro de Testes** | Registro de testes (elétrico, hidráulico etc.) |
---

## 🚀 Como Rodar o Projeto

### Pré-requisitos

- [Node.js](https://nodejs.org/) instalado na máquina (versão 16 ou superior)

### Passo a passo

```bash
# 1. Clone o repositório
git clone https://github.com/joao-luis-0/AV2.git

# 2. Entre na pasta do projeto
cd aerocode

# 3. Instale as dependências
npm install

# 4. Inicie o servidor
npm start
```

Acesse no navegador: **http://localhost:3000**

> Para fazer login, basta digitar qualquer email e senha.
